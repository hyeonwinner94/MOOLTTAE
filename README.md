# 물때 - 낚시 포인트 지도

대한민국 바다 낚시 포인트와 검증된 자연 민물 노지 포인트를 지도에서 탐색하는 정적 웹앱입니다.

## 실행

별도 빌드 없이 로컬 서버로 실행할 수 있습니다.

```powershell
python -m http.server 4173
```

브라우저에서 `http://localhost:4173`을 엽니다.

## 공유 웹사이트와 APK

`main` 브랜치를 GitHub 저장소에 push하고 Pages의 배포 원본을 GitHub Actions로 설정하면 `.github/workflows/deploy-pages.yml`이 정적 사이트를 배포합니다.

Android 디버그 APK는 Android Studio의 내장 JDK와 로컬 Android SDK가 설치된 Windows 환경에서 다음 명령으로 다시 만들 수 있습니다.

```powershell
powershell -ExecutionPolicy Bypass -File tools/build-apk.ps1
```

결과 파일은 `release/moolttae-debug.apk`입니다. 정적 호스팅에 바로 올릴 웹 압축본은 `release/moolttae-web.zip`에 함께 생성됩니다. Play Store 또는 외부 배포용 정식 APK/AAB에는 별도 서명 키가 필요합니다.

## 포함 기능

- 지역, 어종, 검색어 기반 포인트 필터
- 주요 어종 빠른 선택과 정규화된 전체 어종 자동완성 검색
- 데스크톱에서 검색·필터 패널과 결과 목록 패널을 분리한 탐색 레이아웃
- 검색·필터 패널과 결과 목록 패널을 각각 독립적으로 접고 펼치는 지도 확장 기능
- 방파제·방조제, 섬·갯바위, 항구·선착장 등 유형 필터
- 유형, 수심, 진입 특성을 바탕으로 정규화한 초급·중급·숙련 난이도 필터
- 루어, 원투, 찌낚시, 벵에돔, 감성돔, 에깅, 볼락/라이트게임, 농어, 선상/카약 낚시법별 포인트 후보 추천
- 월별 시즌 대상어 필터와 출처 확인 월 또는 후보 어종 4종 이상 기준의 베스트 포인트 필터
- 바낚포, DanFish, 공공데이터 출처 필터
- 난이도별 지도 마커와 포인트 상세 패널
- `localStorage` 기반 즐겨찾기
- 브라우저 위치 권한을 이용한 현재 위치 이동
- 모바일 반응형 레이아웃
- 모바일 상태바 안전 여백, 바다·민물·가이드 탭, 필터·목록 독립 접기
- 지도 이동 후 중심 좌표 기준 Open-Meteo 기상·해상 파고 자동 갱신
- 바다 포인트와 민물 노지 큐레이션 모드 전환
- `guide.html` 별도 화면에서 매듭법, 채비 기초, 대상어별 바다·민물 채비, 출조 준비를 자체 SVG 도식과 단계별 설명으로 확인하는 낚시 가이드
- 가이드 항목별 한국어 YouTube 검색 링크. 특정 영상 삭제에 영향을 받지 않도록 검색 결과를 새 창으로 연결

## 데이터 범위

- 선상 바다낚시 지점 16개: 국립해양조사원 생활해양예보지수 공개 데이터
- 바낚포 전국 카탈로그 397개: 좌표, 포인트명, 시즌, 채비, 수심, 미끼, 어종, 교통 및 특징을 공개 상세 페이지에서 수집한 데이터
- DanFish 바다 카탈로그 1,110개: 사이트맵의 공개 상세 페이지에서 갯바위, 방파제, 항구, 선착장, 해변 유형을 분류해 수집한 데이터

현재 원천 데이터는 총 1,523개입니다. 수동 입력 좌표가 포함된 직접 조사 및 초기 큐레이션 레이어는 지도 정확성을 위해 제외했습니다. 같은 포인트가 서로 다른 카탈로그 출처에 존재할 수 있으므로 출처별 원문을 보존합니다.

바낚포 카탈로그는 `python tools/import-banakpo.py`로 다시 수집할 수 있습니다. 각 상세 화면에 원문 링크가 표시됩니다.

DanFish 카탈로그는 `python tools/import-danfish.py`로 다시 수집할 수 있습니다. 목록은 성능을 위해 최대 180개씩 표시하며 검색과 필터로 범위를 좁힐 수 있습니다.

공식 데이터 출처: [공공데이터포털 - 해양수산부 국립해양조사원 생활해양예보지수 지점 목록](https://www.data.go.kr/data/15145389/fileData.do)

대한민국의 모든 비공식 갯바위, 방파제 포인트를 망라한 단일 공식 데이터셋은 아닙니다. 실제 출조 전에는 기상, 물때, 현장 출입 가능 여부를 별도로 확인해야 합니다.

낚시법별 후보 추천은 MEIS 해양환경 정보지도에서 제공되는 연안주제도, 갯벌, 연안항, 국가어항, 국가해양생태계종합조사 등 공개 레이어 구조를 참고하고, 앱에 저장된 포인트별 유형·수심·바닥 재질·채비·어종 텍스트를 점수화해 계산합니다. MEIS 지도 자체 안내처럼 지도 정보는 법적 효력이 없는 참고 자료이므로, 추천 결과는 출조 전 현장 확인을 대체하지 않습니다.

## 민물 큐레이션

민물 모드는 유료·관리형·인공 낚시터와 댐 관측소 기준점을 지도에 노출하지 않습니다. `freshwater-spots.js`, `national-freshwater-spots.js`, `daeho-freshwater-spots.js`에는 공개 조황 원문과 OSM 수역·교량·역·휴게소 좌표를 교차 확인한 자연 노지 큐레이션 54개가 포함되어 있습니다.

추가로 `osm-freshwater-spots.js`에는 OSM에서 명칭과 좌표가 확인된 전국 자연 수역 탐색 후보 650개가 포함됩니다. 탐색 후보는 조황과 실제 연안 진입로가 아직 교차 확인되지 않았으므로 지도와 카드에서 큐레이션보다 작고 옅게 표시합니다.

전국 탐색 후보는 `python tools/import-osm-freshwater.py`로 다시 수집할 수 있습니다. 포인트 숫자를 늘리기 위해 위치가 모호한 항목을 큐레이션으로 자동 승격하지 않습니다.

민물 마커는 상세 패널에서 정밀도를 구분합니다.

- `진입 기준점`: 교량, 역, 휴게소, 수변 출입로처럼 초행 탐색에 쓸 수 있는 좌표
- `수역 기준점`: 저수지, 하천, 강의 대표 좌표이며 실제 연안 진입로는 현장에서 추가 확인해야 하는 좌표
- `수역 탐색 기준점`: OSM에서 이름과 좌표만 확인된 수역이며 조황과 연안 접근은 아직 교차 확인 전인 후보

후보 대기열은 `python tools/import-freshwater-candidates.py`로 다시 수집할 수 있습니다. 이 도구는 [낚시포인트 민물 게시판](https://fishingpoint.kr/fwfishing)의 공개 게시물에서 위치 단서가 있는 자연 노지 후보를 `data/freshwater-candidates.json`에 저장합니다. 후보 파일은 지도에 직접 연결하지 않습니다.
